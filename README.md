# UILib

#### 项目介绍
Windows平台基于ATL的DirectUI界面库，可以只改1个UIWindow.h文件就能转为基于MFC等，每个控件代码都非常精简，而且性能高、使用简单、功能强大，可以很容易实现任何界面，比DuiLib更好用。
这是我从2012年开始自创的界面库，最开始只有Button控件，根据项目需要不断添加新控件，总共用它做了4个大型客户端产品。
